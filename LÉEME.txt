OOP API Nabla es una adaptación del paradigma de programación orientado a objetos.
Incluye funcionalidades como definir atributos, funciones, herencia y polimorfismo.
Es compatible con tu estilo de programación ya que sus funcionalidades se ejecutan de manera independiente.

Cómo programar orientado a objetos?
Para aprender, sigue el tutorial "Como crear una clase".
En este tutorial se explica paso a paso como crear una nueva clase.
También mira los ejemplos en tu idioma respectivo, éstos ejemplos puedes abrirlo
con un bloc de notas y pegar el código en el editor de un programa o aplicacion de la calculadora HP Prime.

Ten en cuenta que ésta es una primera versión, los bugs y errores se irán corrigiendo eventualmente.
